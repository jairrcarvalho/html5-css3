Principais formatações

Negrito / Itálico

Para o texto ser escrito em segrito usa-se a tag strong.

Já o texto em itálico mostra é mostrado com ênfase e por isso usa a tag em(emphasis em ingles).

Texto marcado

Para marcar um texto utilizamos a tag ,mark, assim o texto irá ficar riscado como quando usamos um marcador de texto.

Texto deletado

Quando queremos mostrar que um texto foi excluído da página utilizamos a tag del , essa tag passa uma linha no meio das letras, o texto ainda pode ser lido mas não deve ser considerado.

Testo inserido

Essa tag é termo inserido no contexto para preencher uam lacuna,deixando-o sublinhado.

Texto grande e pequeno

Utilizamos a tag big no estando essa tag está obsoleta.

Já o texto pequeno ainda se utiliza a tag smoll para diminuir o tamaho da fonte

Texto sub-escrito

Exemplo se texto sub escrito é quando queremos colocar uma fórmula como a da água, usa-se sa tag sub: H2O

Texto sobre escrito

Esse texto também é pra se utilizar uma fórmula mas para situações onde queremos elevar um numero ou letra na parte superior usa-se a tag sup: x2+10
